# Architecture

This project is a standalone authentication system called SecureGate.

Stack:
- Next.js 14 App Router
- TypeScript
- Prisma
- PostgreSQL
- NextAuth
- Resend
- TailwindCSS

Core features:
- Register
- Login
- Email verification
- Forgot password
- Reset password
- Protected dashboard
- Rate limiting

Folder structure:
- src/app for routes
- src/lib for utilities
- src/components for reusable UI
- src/actions for server actions
